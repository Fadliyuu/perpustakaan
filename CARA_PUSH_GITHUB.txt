================================================================================
                    CARA PUSH KE GITHUB - PALING MUDAH
================================================================================

KARENA GIT COMMAND LINE TIDAK JALAN, PAKAI GITHUB DESKTOP!

================================================================================
                        LANGKAH-LANGKAH
================================================================================

1. DOWNLOAD GITHUB DESKTOP
   ------------------------
   Buka: https://desktop.github.com/
   Klik "Download for Windows"
   Install dengan default settings

================================================================================

2. LOGIN GITHUB DESKTOP
   ---------------------
   - Buka GitHub Desktop
   - Klik "Sign in to GitHub.com"
   - Login dengan akun GitHub Anda
   - Jika belum punya akun, daftar di: https://github.com/signup

================================================================================

3. ADD REPOSITORY
   ---------------
   - Di GitHub Desktop, klik "File" → "Add Local Repository"
   - Klik "Choose..." 
   - Pilih folder: C:\Users\Faddd\OneDrive\Documents\web karin
   - Klik "Add Repository"

================================================================================

4. CEK FILE YANG AKAN DI-COMMIT
   ------------------------------
   Setelah add, GitHub Desktop akan menampilkan semua file.
   
   PASTIKAN file berikut TIDAK MUNCUL:
   ❌ .env (di backend atau frontend)
   ❌ backend/firebase-service-account.json
   ❌ node_modules/ (folder besar)
   
   File yang HARUS MUNCUL:
   ✅ backend/env.txt
   ✅ frontend/env.txt
   ✅ Semua file code (.js, .jsx, .css, dll)
   ✅ README.md
   ✅ DEPLOY_CEPAT.md
   ✅ dll

================================================================================

5. PUBLISH KE GITHUB
   ------------------
   - Di GitHub Desktop, klik tombol "Publish repository" (pojok kanan atas)
   - Name: perpustakaan-yp-tunaskarya
   - Description: Sistem Perpustakaan Digital SMK Swasta Tunas Karya
   - ✅ Centang "Keep this code private"
   - Klik "Publish Repository"
   
   Tunggu beberapa detik... Repository akan dibuat di GitHub!

================================================================================

6. COMMIT & PUSH
   --------------
   - Di bawah, ada kotak "Summary"
   - Ketik: "Initial commit - Sistem Perpustakaan Digital"
   - Klik "Commit to main"
   - Setelah commit, klik "Push origin" (pojok kanan atas)
   - Tunggu upload selesai
   
   ✅ SELESAI! File sudah di GitHub!

================================================================================

7. VERIFIKASI DI GITHUB
   ---------------------
   - Buka: https://github.com/YOUR_USERNAME/perpustakaan-yp-tunaskarya
   - Cek file sudah ter-upload
   - Pastikan file sensitif (.env, firebase-service-account.json) TIDAK ADA

================================================================================

SETELAH PUSH KE GITHUB:
-----------------------
1. Deploy Backend ke Railway:
   - Railway → Deploy from GitHub → Pilih repo
   - Setup environment variables dari RAILWAY_VARIABLES.txt

2. Deploy Frontend ke Vercel:
   - Vercel → Import GitHub repo → Pilih repo
   - Root Directory: frontend
   - Setup environment variable dari VERCEL_VARIABLES.txt

================================================================================

TIDAK PERLU GIT COMMAND LINE!
GitHub Desktop akan handle semua untuk Anda!

================================================================================
